This is the reincarnation of a project I did in 2009, to learn PHP and get acquainted with Twitter API. Back then it calculated trending topics for Brazil before Twitter itself deployed regionalized trends.

This new version, albeit using the "old school" version of the trending topics calculation (amount of mentions), has the added advantage of not being bound to a particular city: you can use it with your neighbourhood or point it to an event venue, for example.
